Funções Modulares


Definição:


Função modular é a função que associa elementos de um conjunto em módulos. O módulo é representado entre barras e seus números são sempre positivos, ou seja, mesmo que um módulo seja negativo seu número será positivo. Quando temos uma variável dentro do módulo, precisamos considerar que o valor numérico desta variável pode ser positivo ou negativo. Assim:

|x| = 3

--> x = +3

--> x = -3

Exemplo de equação modular: |x + 1| = 2

x + 1 = 2 --> x = 1

x + 1 = -2 --> x = -3

Gráficos:


Ao representar um módulo negativo, o gráfico para na intersecção é refletido para o sentido positivo.


Exemplos de possíveis gráficos e equações:

  1. Variando o coeficiente do x. Ex: f(x) = |2x|
  2. Somando e subtraindo valores numéricos da função. Ex: f(x) = |x| + 2
  3. Somando e subtraindo valores numéricos dentro do módulo. Ex: f(x) = |x - 1|
  4. Função Modular de uma função do 2º grau. Ex: f(x) = |x² - 2| + 3

    • Como construir esse gráfico manualmente:

    1. Fazer o gráfico da função que está dentro do módulo.
    2. Refletir a parte que tem y negativo para y positivo.
    3. Se tiver algum número somando ou subtraindo fora do módulo, mover o gráfico para cima ou para baixo.
